# 🔐 Secure Document Locker

A simple web application to encrypt and decrypt PDF files using password protection.

## 🚀 Features

- Upload PDF files
- Encrypt with password
- Decrypt protected PDFs
- Download processed files

## 🛠️ Tech Stack

- Python
- Streamlit
- PyPDF2

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
